Testimonial & Refleksi Kelas

Selama mengikuti mata kuliah Pengembangan Web, saya mulai memahami bahwa proses membangun website itu bukan hanya soal menuliskan kode, tetapi juga tentang bagaimana membuat tampilan dan pengalaman pengguna menjadi lebih teratur serta mudah dipahami. Banyak konsep baru yang awalnya terasa asing, namun perlahan mulai saya kuasai.

Salah satu hal yang paling menarik bagi saya adalah ketika mulai menggunakan Tailwind CSS. Cara kerjanya yang berbasis utility class membuat saya lebih cepat bereksperimen dengan desain. Walaupun di awal terasa membingungkan, saya akhirnya menyadari bahwa Tailwind memberi fleksibilitas yang besar dalam mengatur layout dan tampilan tanpa harus membuat file CSS panjang.

Pengalaman mengerjakan proyek akhir ini membuat saya lebih percaya diri untuk mengembangkan website di masa depan. Saya belajar bahwa setiap elemen kecil dalam sebuah halaman web memiliki alasan dan fungsinya masing-masing. Dengan pemahaman ini, saya merasa selangkah lebih dekat untuk menjadi pengembang web yang lebih terampil dan kreatif.
